# aplicativo-teste

Requires PHP 7.4


# Build Setup

 - Enable php.ini extensions:

    - extension=mysqli


 - Install dependencies
    - $ composer update

 - Create .env file
    - cp .env.example .env [Linux]
    - copy .env.example .env [Windows]
# serve at localhost:80
    $ cd path/to/your/app php -S 127.0.0.1:80
